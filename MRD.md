## 一、这个产品是什么

**日光盒子**是一个装在手机里的「心灵疗愈工具箱」。它看起来像一个安静的网页，但你点它、长按它、滑动它，会发现里面藏着好几个不同的房间——

- 有一个房间会随机给你一句励志语录，配上高清的自然风景图
- 有一个房间像一个仪式空间，红⭕️涟漪+音乐+飘落的粒子
- 有一个房间可以和「精神领袖」对话，记录你的心事
- 有一个房间能记录你每天的心情和习惯，像日记本一样
- 还有一个房间是番茄钟+白噪音，帮你进入专注状态

所有这些功能都打包在一个网页里，不用安装 App 就能用，也能一键变成手机桌面应用（PWA），甚至可以导出成安卓 APK。

## 二、为什么要做这个产品

### 2.1 背景：现代人的「隐形内耗」

很多人每天都很累，但说不清为什么累。不是体力活太多，而是——

- 脑子不停地想事情，停不下来
- 刷手机刷到凌晨，早上起不来
- 做事提不起劲，但又很焦虑自己什么都没做
- 心情像坐过山车，不知道为什么突然低落

这些被称为「精神内耗」。它不是病，但持续下去会让人越来越疲惫。

### 2.2 日光盒子的回答

日光盒子想做的是：**在你不好的时候，拉你一把，给你一个安静的角落**。

它不是心理医生，不是课程平台，也不是社交软件。它就像一个朋友，在你需要的时候——

- 给你一句鼓励的话
- 放一首让你平静的音乐
- 陪你专注 25 分钟
- 帮你把今天的心情记下来

产品核心理念只有一句话：

> **纵陷至暗，自愈伤痕。Even in the dark, you can heal my scars.**

## 三、给谁用

| 人群 | 典型场景 |
|------|---------|
| **大学生** | 期末考试焦虑，论文写不出，晚上睡不着；打开日光盒子做 25 分钟番茄钟，听雨声白噪音，做完心情好一点 |
| **职场新人** | 工作压力大，情绪波动频繁，没人可以倾诉；进精神领袖模块写几句，回去继续干活 |
| **自由职业者** | 在家办公缺乏节奏感，容易拖延；用番茄钟+每日打卡建立规律 |
| **焦虑/情绪困扰者** | 晚上 9 点焦虑高发时段，看到太阳入口亮起，进去写几句话，完成一次「焦虑移交」仪式 |

> 目标用户年龄：16–35 岁，以 Z 世代为主。对心理健康的接受度高，愿意尝试非传统疗愈方式。

### 4.5 设计哲学：三段式冥想入口（Triadic Meditative Portal）

你可能注意到了，从主页出发，只有 **3 条路**可以走：

| 操作 | 目的地 | 代表意义 |
|------|--------|---------|
| 1 秒内 3 击太阳 | 励志页面 | **励（Inspire）** |
| 7 连击隐藏触发位 | 精神领袖 | **省（Reflect）** |
| 点右下角 🌱 | 身心恢复 | **行（Act）** |

这不是随便放了 3 个按钮，而是一个完整的心理疗愈闭环。它的逻辑和东方智慧一脉相承：

#### 为什么恰好是 3？

道教讲「三生万物」，佛教讲「戒定慧」——三步递进，缺一不可。

日光盒子的 3 条路正好对应一个人从低谷走出来的完整过程：

**第一步：励（Inspire）—— 被触动**
当你低落的时候，首先需要的不是方法，而是一点亮光。一句「沉潜不躁，蓄力不怠」出现在红色光晕里，配上自然风景和背景音乐 —— 它不教你该怎么做，它只是陪着你，让你感觉不那么孤单。

**第二步：省（Reflect）—— 向内看**
情绪被触动之后，才有力气面对自己。在精神领袖的界面里，你对着那个蓝色光环的太阳打字。不是在跟别人倾诉，而是在跟自己对话。每一条记录都被保存下来，你可以回头翻看 —— 看到自己曾经怎么走过来的。

**第三步：行（Act）—— 付诸行动**
觉察之后，需要一个出口。番茄钟帮你进入专注，雨声白噪音盖住噪音，呼吸练习让你平静下来，心情打卡帮你建立记录的习惯。这些不是「你应该做的」，而是「你想做的时候，它们刚好在」。

#### 为什么不做 4 个、5 个？

超过 3 个入口会让人产生选择焦虑（「我该点哪个？哪个功能最好用？」），反而偏离了「放下思考，自然交互」的初衷。少于 3 个则链条断裂 —— 比如只有激励没有行动，就沦为鸡汤；只有打卡没有激励，就变成冰冷的任务管理。

> **3 条路，一个闭环：被触动 → 自我觉察 → 付诸行动。每一步都在前一步的基础上自然发生。**

## 五、模块一：太阳主页

### 5.1 这是什么

用户打开应用看到的第一画面：深蓝色背景中央，一颗金红色的太阳安静地亮着，周围隐约能看到细密的日冕光芒。

### 5.2 怎么玩

| 操作 | 条件 | 效果 |
|------|------|------|
| **单击太阳** | 随时 | 太阳震动一下，向外弹出一圈红色涟漪 + 金色粒子 |
| **三连击** | 1 秒内点 3 次 | 金环从太阳向外扩展开来，屏幕闪过白光，进入**语录全屏页** |
| **长按** | 按住超过 220 毫秒 | 太阳逐渐变暗，周围出现「日食」光晕 |
| **超长按** | 按住超过 3 秒 | 光芒万丈爆发——日冕射线炸开，持续约 2 秒后自动消失 |

### 5.3 为什么要这样设计

- **视觉反馈给用户安全感**：每一次点击都有明确反馈（涟漪、粒子），不会觉得「没点到」
- **操作有层次**：轻点 → 三击 → 长按 → 超长按，对应越来越深的功能，像剥洋葱
- **不设文字按钮**：整个主页除了右下角一个小 🌱 图标，没有任何文字。目的是让用户放下思考，像玩一个解压玩具一样自然交互

### 5.4 技术细节

- 太阳光晕用 Canvas 绘制，含多层 `conic-gradient` 合成日冕纹理
- 粒子系统直接在 DOM 上操作，每个粒子有独立的物理轨迹（速度、角度、透明度衰减）
- 三连击判断用时间窗口算法：记录每次点击时间戳，刷掉 1 秒之外的旧记录

## 六、模块二：励志页面

### 6.1 这是什么

进入后看到的第一个「内页」。黑色背景中央，一个红色的光圈在缓缓呼吸，光圈外有涟漪一波一波向外扩散（像下雨天湖面的水波），光圈的碎片层缓慢旋转着。

光圈正中央是一句励志语录（中英文双行）。底部有精神纲领小字。背景随机展示一张高清自然风景图。

### 6.2 核心元素

#### 6.2.1 红色光圈（主体）

- **外层环**：呼吸式缩放（慢速，约 5 秒一个周期），不再快速旋转，而是温和地一收一放
- **涟漪波**：外层环的 `::before` / `::after` 伪元素以不同时间差向外扩散，营造「雨滴入湖」的涟漪感
- **碎片层**：三层 `repeating-conic-gradient` 叠加，不同速度旋转，生成科技感的碎片光环
- **闪光层**：`conic-gradient` 长条闪光，配合 `blur` 和 `mix-blend-mode: screen`，有「能量流动」的感觉

#### 6.2.2 背景音乐

每次进入页面，从下面两首中**随机**挑一首：

| 曲目 | 文件 | 大小 |
|------|------|------|
| 《The Cage》 | `the_cage.mp3` | 7.3MB |
| 《Ping Pong 2》 | `pingpang2.mp3` | 4.6MB |

音乐支持淡入淡出，不会突然炸耳。如果浏览器禁止自动播放，第一次点击屏幕时才会开始。

#### 6.2.3 语录池

所有语录存在 `whisper-quotes.json` 里，当前 12 条，例如：

| 中文 | 英文 |
|------|------|
| 疾如风，徐如林，侵掠如火，不动如山 | Swift as the wind, quiet as the forest... |
| 沉潜不躁，蓄力不怠；时机一至，雷霆万钧 | Dive deep without restlessness... |
| 向上，是冲破一切黑暗的终极钥匙 | Upward is the master key through all darkness... |

语录每次随机选取，同一次会话里不重复（用 `sessionStorage` 记忆最近 6 条）。

#### 6.2.4 背景图

从 `public/whisper/` 文件夹的 5 张高清自然图中随机选一张（1920×1280 分辨率），去重逻辑同语录。

#### 6.2.5 特效粒子

- **火星**：30 颗红色光点从顶部向下飘落，大小随机，模拟灰烬或萤火虫
- **灰烬**：15 片灰色不规则碎片，带旋转飘落
- **羽毛**：偶尔生成红色羽毛状粒子，慢速飘落
- **凤凰**：7 连击隐藏按钮触发后，一只红色凤凰从左侧飞过画面

#### 6.2.6 点击反馈

在页面上任意点一下，会产生红色涟漪向外扩散 + 小火花四散。

### 6.3 底部小字

`向上，是冲破一切黑暗的终极钥匙`

### 6.4 右上角切换按钮

点一下可以切换「战场模式」和「柔和模式」，调整光圈的旋转速度和光晕强度。

## 七、模块三：精神领袖

### 7.1 入口

在励志页面中央的上方有一个完全透明的触发按钮。在 1 秒内快击它 7 次，屏幕会逐渐变暗，然后进入精神领袖页面。

（现在右上角也有一个 📋 按钮，可以直接进入，不用 7 连击）

### 7.2 这是什么

一个很安静的对话界面。屏幕中央有一个蓝色光环的微型太阳，下方是输入框。

用户可以在这里**和自己对话**。输入任何文字，它会像消息气泡一样出现在上方，背景色是淡淡的金色。

### 7.3 对话流程

1. 输入文字，按回车发送
2. 文字出现在对话区
3. 发送一定数量后，屏幕中央浮现一个身影轮廓
4. 屏幕显示出固定标语：**「纵陷至暗，自愈伤痕」**（中英文）
5. 太阳光环开始闪烁

### 7.4 历史记录

- 点右上角 📋 按钮 → 进入**历史记录列表**
- 按日期分组展示
- 每条记录显示日期和第一句话的预览
- 点某一天 → 查看当天全部对话内容
- 底部有 🔥 按钮，可以删除全部历史（带确认提示）

数据存在 `localStorage`，纯本地存储，不会上传到任何服务器。

### 7.5 背景音乐

`leader-theme.mp3`（2.1MB），循环播放，能量波动画随音乐呼吸。

## 八、模块四：焦虑移交

### 8.1 这是什么

一个**时间限定的隐藏功能**。

每天 **21:00 到 21:59**，在励志页面上会出现一个金色小太阳图标。点击它会进入「焦虑记录」页面。

### 8.2 设计理念

晚上 9 点是一天中焦虑感最容易上来的时段。在这个时间窗口，用户可以把当天的焦虑「移交」给太阳——写下来，然后放下。

### 8.3 功能

- 输入框：写下今天让你焦虑的事情
- 提交后：太阳图标消失，代表「焦虑已移交」
- 可查看历史记录（焦虑记录 + 精神领袖对话按天混排）

### 8.4 为什么是隐藏入口

不主动展示，不设推送提醒。只在那个时间点悄悄亮起一个图标。这样做的目的是**不给用户制造压力**——你不是必须去记录，而是在你需要的时候，它刚好在。

## 九、模块五：身心恢复

### 9.1 入口

主页右下角 🌱 按钮 → 点击产生水波扩散动画 → 进入恢复页面。

### 9.2 三个标签页

| 标签 | 图标 | 功能 |
|------|------|------|
| 打卡 | ✅ | 每日心情 + 习惯 |
| 专注 | 🌳 | 番茄钟 + 白噪音 + 呼吸练习 |
| 记录 | 📊 | 数据图表 + 成就系统 |

### 9.3 打卡功能

#### 心情选择
5 个表情，从 😞 到 😌，对应 1-5 分。

#### 习惯打勾
6 项每日习惯：

- 早睡早起
- 三餐规律
- 多喝水
- 散步
- 阅读
- 冥想

每完成一项打勾。

#### 今日评分
根据心情 + 习惯完成数 + 是否专注过，算出一个百分制分数。

### 9.4 记录功能（数据图表）

| 图表 | 内容 |
|------|------|
| 心情曲线 | 过去 7/14/30 天情绪变化 |
| 习惯热力图 | 连续打卡天数 |
| 番茄钟统计 | 每日专注分钟数 |
| 呼吸训练 | 呼吸轮数趋势 |
| 成就系统 | 17 个成就徽章（如「连续打卡 365 天」「累计专注 100 个番茄」等） |

### 9.5 数据安全性

所有数据存储在浏览器本地（`localStorage` + `IndexedDB`），**不经过任何服务器**。你的心情记录、习惯数据、对话内容，都只在你自己的手机里。

### 9.6 呼吸练习

在恢复页面的「专注」标签里。

- 三个速度档位：慢（6-3-8 秒）、标准（4-2-6 秒）、快（3-1-4 秒）
- 一个圆圈随着吸气膨胀、呼气缩小
- 完成一轮后记录数据
- 底部显示今日呼吸训练总时长

## 十、模块六：番茄专注

### 10.1 这是什么

一个带**树木成长动画**的番茄钟。

### 10.2 模式选择

| 模式 | 专注 / 休息 | 说明 |
|------|------------|------|
| 深度专注 | 25+5 | 经典 Pomodoro |
| 长时专注 | 45+10 | 适合写代码/论文 |
| 短时冲刺 | 15+3 | 快速进入状态 |
| 自定义 | 自定 | 自己设置分钟数 |

### 10.3 树木成长可视化

- **种子阶段**：一个小球埋在土里，带呼吸动画
- **发芽阶段**：一根绿芽从土里冒出来
- **幼苗阶段**：主干长高，第一层树冠出现
- **小树阶段**：树冠扩大，长叶子
- **枝繁叶茂**：36 片叶子 + 多层树冠
- **结果阶段**：休息时间显示 6 颗红色果实挂在树冠上

全部用 **纯 SVG** 绘制，零图片依赖。

### 10.4 白噪音

开始专注时自动循环播放 **「龙猫森林雨」**（`totoro-rain.mp3`，19MB，真实雨声录音）。暂停/重置时停止。

可手动关闭。

### 10.5 专注结束

- 播放三和弦提示音（C5-E5-G5，Web Audio API 合成）
- 自动进入休息阶段
- 休息结束后显示「已完成」，可点「🌱 再来一棵」

### 10.6 统计数据

- 今日几棵🌳（累计专注分钟 ÷ 25）
- 累计专注总分钟数

## 十一、技术实现亮点

### 11.1 纯前端，零后端

整个应用没有任何后端服务器。所有数据存在浏览器的 localStorage 和 IndexedDB 里。不需要注册账号，不需要登录，打开就能用。

### 11.2 React + 原生 DOM 混合架构

- React 负责页面路由和组件生命周期
- 核心动画引擎（`PhoenixPage.js`，770+ 行）用纯原生 JavaScript 实现，零动画库依赖
- 太阳光晕用 Canvas 绑定绘制
- 树木成长用 SVG 绘制

### 11.3 AI 协作开发

本项目通过和 AI 对话迭代完成，体现现代软件工程的 **Prompt Engineering** 和 **AI-Native 开发**能力。人类负责产品设计和决策，AI 负责代码实现和问题排查。

### 11.4 核心数字

## 十二、跨平台分发

### 12.1 Web

部署在 Netlify，绑定自定义域名 `sunlight.qinhaoyi.xyz`，自动 HTTPS。

### 12.2 PWA（Progressive Web App）

- 配置了 `manifest.webmanifest`，支持添加到手机主屏幕
- Service Worker 自动缓存全站静态资源，离线可用
- 表现为独立窗口，没有浏览器地址栏

### 12.3 TWA（Trusted Web Activity，安卓 APK）

通过 Google 的 `@bubblewrap` 工具，将 PWA 打包成原生安卓 APK：

- APK 文件大小约 2MB（WebView 壳）
- 安装后首次打开自动加载线上资源
- Service Worker 缓存后离线也能用
- 可上架 Google Play
