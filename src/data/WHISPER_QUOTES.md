# Sun Whisper 语料手册

三连击太阳后显示的文案，**只从这里读取**。

## 编辑哪个文件

**`src/data/whisper-quotes.json`**

改完保存，刷新页面（开发时 `npm run dev` 会自动热更新）。

## 每条格式

```json
{
  "id": "my-quote-01",
  "en": "English line shown above (optional, can repeat zh or leave short)",
  "zh": "中文主文案，屏幕上较大一行"
}
```

| 字段 | 说明 |
|------|------|
| `id` | 唯一英文短名，勿重复，如 `upward-01` |
| `zh` | **主文案**（金色大字，屏幕上最醒目） |
| `en` | 可选副文案（较小、在中文下方）；不需要时写 `""` 即可只显示中文 |

## 新增一条（复制粘贴）

在 JSON 数组里**最后一条后面加逗号**，再粘贴：

```json
,
{
  "id": "new-quote",
  "en": "Your English line",
  "zh": "你的中文句子"
}
```

## 删除一条

删掉对应 `{ ... }` 整块；注意前后逗号，保证仍是合法 JSON（可用编辑器格式化检查）。
